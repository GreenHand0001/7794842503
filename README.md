# 项目简介

## 核心功能
- 支持图片、视频及摄像头进行无人机检测
- 提供UI界面，实时显示目标位置、目标总数和置信度信息
- 支持批量检测并在界面直接查看所有检测结果
- 支持检测结果保存
- 可选择单张图片或图片文件夹进行识别
- 可选择视频文件进行识别
- 可调用本地摄像头进行实时识别
- 支持自定义置信度和IOU阈值
- 可选择是否显示标签和原图
- 可选择使用不同检测模型
- 包含1300张无人机标注数据集(drone，1类)

## 适用场景
- 无人机监控与安全管理
- 空域安全监测
- 无人机活动追踪与研究
- 安全区域入侵检测

## 技术特点
- 基于YOLOv8深度学习框架开发
- 使用Python3.9、OpenCV、PyQt5等技术栈
- 集成Ultralytics和Torch1.9深度学习库
- 提供完整训练好的模型及训练结果
- 配置成功环境可直接使用，无需重新训练
